# Full-Stack-E-Commerce-MERN-APP
Full Stack E-Commerce MERN APP


Full Stack E-commerce Application
This project is a complete, feature-rich MERN Stack E-commerce website built using React.js, MongoDB, Express.js, and Node.js.

Key Features:
User Authentication: Secure sign-up and login functionality .
Product Discovery: Browse, filter, and sort products with a dynamic product gallery (4:16-4:51, 2:03:06).
Shopping Cart: Add products with specific variants like size and manage your cart items (5:03-5:10, 3:41:41).
Checkout & Payments: Integration with Stripe and Razorpay for secure online transactions, alongside a Cash on Delivery option.
Admin Dashboard: A dedicated backend panel for administrators to manage inventory, upload products, and track orders .
Deployment: Configured for seamless deployment on Vercel.
Tech Stack:
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Storage: Cloudinary (for image hosting)
Payment Gateways: Stripe, Razorpay
How to Get Started:
Clone the repository: git clone [your-repo-link]
Setup Frontend: Navigate to the frontend directory, install dependencies (npm install), and run the development server (npm run dev) .
Setup Backend: Connect your MongoDB database and configure environment variables.
Deployment: Ensure vercel.json configurations are set for both frontend and backend to support routing and API calls .
